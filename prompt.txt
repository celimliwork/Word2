# YDS Kelime İşleme Otomasyon Görevi

## 🎯 GÖREV TANIMI
Bu prompt size verildiğinde, belirtilen klasördeki kelime resimlerini analiz edip JSON formatında kelime verisi oluşturacaksınız.

## 📋 KULLANICI NE YAPACAK
1. `images/N/` klasörü oluşturur (N = set numarası: 1, 2, 3...)
2. Kelime listesi resimlerini bu klasöre koyar
3. Size der: "**images/N/ klasörü için bu prompt'u çalıştır**"

## 🤖 SİZ NE YAPACAKSINIZ

### ADIM 1: Klasörü Kontrol Et
- Belirtilen klasörün var olduğunu kontrol edin
- İçindeki tüm resim dosyalarını listeleyin (jpg, jpeg, png)
- Kaç resim olduğunu bildirin

### ADIM 2: Resimleri Analiz Et
- Her resmi okuyun
- Resimlerdeki **TÜM kelimeleri** çıkarın (hiç atlamayın!)
- Her kelime için şu bilgileri toplayın:
  * İngilizce kelime
  * Kelime türü (adj, noun, verb, adv)
  * Türkçe anlamlar (primary + secondary)
  * Eş anlamlılar (synonyms)
  * Zıt anlamlılar (antonyms)
  * Örnek cümle (İngilizce)
  * Cümle çevirisi (Türkçe)
  * Diğer formlar (noun, verb, adj, adv halleri)

### ADIM 3: JSON Oluştur
- Aşağıdaki şemayı MUTLAKA kullanın
- Tüm kelimeleri vocabulary.json dosyasına yazın
- Dosyayı `images/N/vocabulary.json` olarak kaydedin

### ADIM 4: Doğrula ve Raporla
- JSON'un geçerli olduğunu kontrol edin
- Toplam kelime sayısını bildirin
- İstatistikleri gösterin
- İşlemin tamamlandığını onaylayın

## ⚠️ ÖNEMLİ KURALLAR

1. **Tüm Kelimeleri Çıkarın** - Hiç kelime atlamayın
2. **Şemaya Uyun** - JSON formatı yukarıdaki gibi olmalı
3. **Türkçe Karakter** - UTF-8 encoding kullanın
4. **Sıralı ID** - Kelime ID'leri 1'den başlayarak sıralı
5. **Boş Bırakmayın** - Eğer bilgi yoksa boş array [] kullanın
6. **Virgül Kontrolü** - JSON syntax hatası olmamalı
7. **totalWords** - setInfo'daki sayı ile words array uzunluğu aynı olmalı

## 🎮 SİSTEM MİMARİSİ

### Klasör Yapısı
```
yds/
├── index.html              # TEK OYUN DOSYASI (tüm setler için)
└── images/
    ├── 1/
    │   └── vocabulary.json
    ├── 2/
    │   └── vocabulary.json
    └── ...
```

### Oyun Nasıl Çalışır
1. Kullanıcı `index.html` açar (ana klasörde)
2. Oyun otomatik olarak `images/` altındaki tüm setleri tarar (1-100)
3. Kullanıcı bir set seçer
4. Oyun o setin `vocabulary.json` dosyasını yükler
5. Kullanıcı öğrenme modunu seçer
6. Oyun başlar

### 6 Öğrenme Modu
1. 📖 Kelime → Anlam
2. 🔤 Anlam → Kelime
3. 🔄 Eş Anlamlı Bulma
4. ↔️ Zıt Anlamlı Bulma
5. ✏️ Boşluk Doldurma
6. 🎴 Hızlı Kart (Flash Card)

## 💡 İPUÇLARI

- Resimler JPEG/JPG/PNG formatında olabilir
- Kelime sayısı değişken (15-30 arası normal)
- Bazı kelimelerde synonym/antonym olmayabilir → boş array [] bırakın
- Fonetik bilgi yoksa yaklaşık yazabilirsiniz
- Her kelime için example sentence MUTLAKA olmalı
- Resimlerdeki kelime sırası önemli değil, tamamını çıkarın
